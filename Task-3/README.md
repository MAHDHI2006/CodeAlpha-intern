# Task 3: Secure Coding Review

## Objective
To identify insecure coding practices and apply secure coding techniques to prevent vulnerabilities.

## Vulnerability Identified
SQL Injection caused by improper handling of user input.

## Tools Used
- Python
- Bandit (Security Analysis Tool)

## Secure Coding Approach
Parameterized queries were used to prevent user input from altering SQL logic.

## Outcome
The vulnerable code was fixed, reducing the risk of SQL injection attacks.
